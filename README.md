# Kolesh Git tutorials!
- git pull
- git push
- git clone
- git stuffs....
- git log: Shows a history of all commits in the current branch.
- git diff: Shows the differences between the current state of the repository and the last commit.
- git reset: Unstages changes in the staging area.
- git revert [commit-hash]: Reverts a specific commit, creating a new commit that undoes the changes.
- git stash: Stashes changes in a temporary location, allowing you to switch to a different branch without committing or losing changes.
- git remote add [name] [url]: Adds a new remote repository with a specified name and URL.
- git fetch: Downloads changes from a remote repository, but does not merge them into the local branch.
- git tag [tag-name]: Creates a new tag at the current commit with a specified name.
- git show [commit-hash]: Shows the details of a specific commit.
- git blame [file]: Shows who last modified each line in a specified file.
